# Supervertaler Workbench

Supervertaler Workbench is a standalone desktop CAT (Computer-Assisted Translation) tool built with Python and PyQt6. It is free and open source (MIT licence) and runs on Windows, macOS, and Linux.

Supervertaler Workbench gives you AI-powered translation, full translation memory, terminology management, quality assurance, and import/export for common file formats — all in one application, without a subscription.

![Supervertaler Workbench](https://supervertaler.com/screenshots/Supervertaler_v1.9.329_Term_and_NT_insert_popup.png)

## Key Features

### AI Translation
Translate single segments or entire files using any major LLM provider — OpenAI, Anthropic, Google Gemini, Mistral, Grok, DeepSeek, and more. Domain-specific prompts and a built-in Prompt Library help you get consistent, high-quality output. Run models locally with Ollama for a fully offline workflow.

### Translation Memory
Store and reuse confirmed translations with fuzzy matching. Import existing TMX files to seed your TM, and let SuperMemory surface the best match automatically.

### Glossaries & TermLens
Create and manage glossaries in Supervertaler's SQLite `.db` format. TermLens highlights matched terms directly in the segment grid so you never miss a term. Extract new terms automatically from your source text.

### Import & Export
Open DOCX, TXT, SDLXLIFF, bilingual tables, and multi-file projects. Export translations back to the original format with one click.

### SuperLookup
A combined lookup panel showing TM matches, glossary hits, machine translation, and web resources side by side — all driven by the current segment.

### Quality Assurance
Built-in spellcheck, tag validation, and non-translatable detection catch common errors before you deliver.
